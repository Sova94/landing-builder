# Landing Builder - Резюме проекта

## 📦 Что создано

### Полный стек конструктора лендингов уровня Tilda

#### Frontend (React + TypeScript + Vite)
- ✅ Визуальный редактор с drag-and-drop
- ✅ Система виджетов (5+ готовых)
- ✅ Панель настроек с live preview
- ✅ Адаптивность (Desktop, Tablet, Mobile)
- ✅ Undo/Redo история изменений
- ✅ Monaco Editor для пользовательского кода
- ✅ UI компоненты (Radix UI + Tailwind CSS)
- ✅ Zustand для управления состоянием

#### Backend (NestJS + PostgreSQL)
- ✅ REST API для всех операций
- ✅ JWT аутентификация
- ✅ TypeORM с PostgreSQL
- ✅ Swagger документация
- ✅ Модульная архитектура
- ✅ Генерация статических лендингов
- ✅ Система шаблонов

#### Ключевые возможности

**Виджеты:**
1. Текст (TextWidget)
2. Заголовки (HeadingWidget)
3. Кнопки (ButtonWidget)
4. Hero блоки (HeroWidget)
5. Custom Code с Monaco Editor

**Редактор кода:**
- Автодополнение для HTML/CSS/JS
- Виджет-подсказки (@form, @button и т.д.)
- Сниппеты (hero, faq, slider, form)
- Валидация в реальном времени
- Контекстные советы

**Публикация:**
- Генерация чистого HTML/CSS/JS
- Поддержка своих доменов
- SEO настройки
- Интеграция аналитики

## 🚀 Как запустить

### Локально:
```bash
npm run install:all
npm run dev
```

### Docker:
```bash
docker-compose up -d
```

Доступ:
- Frontend: http://localhost:3000
- Backend API: http://localhost:4000
- API Docs: http://localhost:4000/api
