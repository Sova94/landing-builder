# Шаблоны лендингов

Библиотека готовых шаблонов для быстрого запуска проектов.

## Категории

### Business
- Корпоративный сайт
- Лендинг продукта
- Landing page услуги
- Презентация компании

### SaaS
- SaaS продукт
- Веб-приложение
- Мобильное приложение
- API документация

### E-commerce
- Интернет-магазин
- Карточка товара
- Каталог
- Landing page акции

### Agency
- Портфолио агентства
- Креативное агентство
- Маркетинговое агентство
- IT-агентство

### Portfolio
- Личное портфолио
- Фотограф
- Дизайнер
- Разработчик

### Education
- Онлайн-курс
- Вебинар
- Образовательная платформа
- Школа

### Real Estate
- Недвижимость
- Квартал
- Офис
- Апартаменты

## Создание нового шаблона

1. Создайте проект в редакторе
2. Соберите страницу из виджетов
3. Экспортируйте проект как шаблон
4. Добавьте мета-данные (название, категория, теги, превью)
5. Зарегистрируйте в системе

## Формат шаблона

```json
{
  "name": "Название шаблона",
  "description": "Описание",
  "category": "business",
  "preview": "https://example.com/preview.jpg",
  "project": {
    "sections": [...],
    "settings": {...},
    "seo": {...}
  },
  "tags": ["корпоративный", "современный", "минимализм"],
  "author": "Имя автора"
}
```
