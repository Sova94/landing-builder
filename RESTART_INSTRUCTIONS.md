# Инструкция по перезапуску после исправлений

## 1. Остановите текущие серверы

Нажмите `Ctrl+C` в обоих терминалах где запущены backend и frontend.

## 2. Очистите кэш и перезапустите

```powershell
# Остановите все node процессы
Get-Process -Name node | Stop-Process -Force

# Перейдите в frontend и очистите кэш
cd landing-builder\frontend
Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue

# Вернитесь в корень
cd ..\..

# Перезапустите backend
cd landing-builder\backend
npm run start:dev
```

**В НОВОМ ТЕРМИНАЛЕ:**

```powershell
cd landing-builder\frontend
npm run dev
```

## 3. Что было исправлено

- Убран middleware `immer` из editorStore
- Упрощены компоненты виджетов (TextWidget, HeadingWidget, ButtonWidget, HeroWidget, CustomCodeWidget)
- Убраны лишние импорты `cn` и сложные типы в Monaco Editor

## 4. Если что-то не работает

### Ошибки в консоли браузера (F12)

1. Откройте http://localhost:3000
2. Нажмите F12 → Console
3. Скопируйте ошибки и отправьте разработчику

### Виджеты не добавляются

1. Убедитесь что выбрана секция (кликните на секцию на канвасе)
2. Попробуйте добавить виджет из левой панели
3. Проверьте консоль на ошибки

### Панель настроек пустая

1. Кликните на виджет на канвасе
2. Панель настроек должна открыться справа
3. Если не открылась - проверьте консоль браузера

## 5. Проверка работоспособности

- [ ] Frontend открывается на http://localhost:3000
- [ ] Backend API на http://localhost:4000/api
- [ ] Можно добавить виджет из левой панели
- [ ] При клике на виджет открываются настройки справа
- [ ] Можно менять настройки и видеть изменения
