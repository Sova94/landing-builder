## 🌐 Как открыть

1. **Backend API**: http://localhost:4000/api
2. **Frontend редактор**: http://localhost:3000

## 🎯 Первый запуск

1. Откройте http://localhost:3000 в браузере
2. Зарегистрируйтесь (или войдите)
3. Создайте новый проект
4. Добавьте виджеты из левой панели
5. Настройте виджеты в правой панели
6. Нажмите "Опубликовать"

## 🛑 Как остановить

**Backend**: В терминале где запущен backend нажмите `Ctrl+C`

**Frontend**: В окне где запущен frontend нажмите `Ctrl+C`

## 🔧 Если нужно перезапустить

```powershell
# Остановить все процессы
Get-Process -Name node | Stop-Process

# Запустить backend
cd landing-builder/backend
npm run start:dev

# В новом окне запустить frontend
cd landing-builder/frontend
npm run dev
```
